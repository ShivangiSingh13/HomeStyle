# ✅ Teacher's SEO Requirements Implementation Guide

## 📋 Meta Tags Character Limits (Per Teacher)

### Page Titles: 50-70 characters
- Keep concise and keyword-focused
- Include primary keyword
- Include brand name where possible

### Meta Descriptions: 155-160 characters
- Clear value proposition
- Include 1-2 primary keywords
- Include CTA when relevant

### H1 Headings: 50-70 characters
- **IMPORTANT:** Put target keyword at the BEGINNING
- Make it descriptive and user-friendly
- One H1 per page only

## 🖼️ Image Naming Convention

### DO ✅
- `ergonomic_study_desk_for_students.jpg`
- `modern_office_chair_leather_black.jpg`
- `wall_art_canvas_set_3_piece.jpg`
- `ceramic_planters_indoor_plants.jpg`
- `LED_desk_lamp_USB_charging.jpg`
- `wooden_bookshelf_storage_unit.jpg`
- `home_office_setup_complete.jpg`

### DON'T ✗
- `image_1345_76.jpg`
- `photo1.jpg`
- `desk.jpg`
- `IMG_2024.jpg`

**Rule:** Use underscores between words, include 2-3 relevant keywords, use lowercase

## 🔧 Tracking Tools to Integrate

### 1. Hotjar (Real-time Visitor Behavior)
- Tracks how users interact with pages
- Heatmaps and session recordings
- Add code to `<head>` section

### 2. InChargeads (Tracking & Ad Optimization)
- One-click tracking solution
- Optimize ads with first-party data
- Improve ROI tracking

### 3. Optimizely (Landing Page Testing)
- A/B test landing pages
- Visual creation tools
- Audience targeting
