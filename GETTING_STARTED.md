## 🛠️ How to Customize

### 1. Basic Information
Edit `contact.html`:
- Change address
- Update phone number
- Modify email
- Change business hours

### 2. Update Products
Edit `js/products-data.js`:
- Change product names
- Update prices
- Modify descriptions
- Add/remove products
- Update ratings

### 3. Change Colors
Edit `css/style.css`:
- Update CSS variables at the top
- Change accent color
- Modify background colors
- Update text colors

### 4. Add Blog Posts
Create new HTML in `/blog/` folder:
- Copy an existing blog post template
- Update content
- Modify meta tags
- Add to blog.html listing

### 5. Replace Images
Add actual product images:
- Create `/assets/images/` folder
- Add product images (JPG, PNG)
- Update image URLs in `products-data.js`
