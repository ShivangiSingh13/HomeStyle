# HomeStyle - Premium Furniture & Home Decor E-Commerce Website

A modern, SEO-optimized e-commerce website built with HTML, CSS, and JavaScript featuring furniture, home decor, and study setup products.

## 📁 Project Structure

```
digital/
├── index.html                 # Homepage with hero section
├── products.html              # Product listing page with filters
├── product-detail.html        # Individual product detail page
├── blog.html                  # Blog listing page
├── contact.html               # Contact page with form
├── blog/
│   ├── best-study-table.html      # Blog: Best study tables guide
│   ├── aesthetic-room-setup.html  # Blog: Aesthetic room ideas
│   └── home-decor-trends.html     # Blog: 2026 decor trends
├── css/
│   ├── style.css              # Main styling
│   ├── responsive.css         # Mobile responsive styles
│   └── cart-and-utilities.css # Cart and utility classes
├── js/
│   ├── script.js              # Main JavaScript functionality
│   └── products-data.js       # Product and blog data
└── assets/                    # Images and media folder
```

## 💾 Data Persistence

The shopping cart is saved to browser's **LocalStorage**, meaning:
- Cart items persist even after refreshing the page
- Cart clears when browser cache is cleared
- Each browser has its own cart
